Introduction

KaeChat is an IRC client coded from scratch in Python, originally made as a learning exercise.

It supports basic IRC client functionality like multiple networks, alerts on highlight, etc, in addition to some niceties like highlighting URLs.

Main Window

The main window consists of the menu bar, and several tabs. The first tab is named (KaeChat), and is used for global messages. The other tabs are Network Tabs; they are like one separate IRC client for each network you connect to. You can connect to a network by selecting Network → Network list... from the menu bar.

Menu Bar

Network Menu

  • Network List...

    Shows the Network List window, from which you can connect to a network, or edit your networks.

  • Disconnect

    Disconnects from the current network, and close the respective tab.

  • Quit

    Disconnects from all networks and quits the program.

Edit Menu

  • Preferences...

    Shows the Preferences window, from which you can edit program configuration.

Help Menu

  • About...

    Shows information about this program.

Network List

The Network List window can be used to edit your networks graphically, or to connect to one of your configured networks.

To the left is a list of currently configured networks. You can use the Add button to add a new network. The Remove button deletes the currently selected network; you will be asked for confirmation, but this operation cannot be undone. The Edit button opens a window to edit the currently selected network. Clicking Close will close the window.

To connect to a network, double click it on the list, or select it from the list and click Connect.
